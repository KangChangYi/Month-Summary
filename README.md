## Month Summary

躺在电影院沙发上，眼前的电影，播放着这一幕：

主角从任意门穿梭到未来，发现自己只是一个没用的打工人，他不甘心，问未来自己为什么这么没用，未来的自己说：“因为你都没好好学习，只顾着娱乐，平时看的书也没发挥什么作用，就你这个样子，还想我现在好到哪里去？”，主角很失望，他回到现实，在脑海中烙下了一段话：“我要成为奋斗逼”。平复了情绪之后，他又再次穿越到未来，但这一次，未来的自己是一个大老板，他很开心，他上前打招呼，表示看到自己未来这么成功非常开心，但未来的自己说：“可是我不开心，你每天拼了命的学习，拼命的完成工作，因为忙于事业两年后，失去了你心爱的人，还把我的身体拖垮了，我现在觉得我活不到 60岁，你开心吗？”，原来拼命赚钱也不会开心啊，主角这样想着回到了现实，他这次下定决心.....

以上情节纯属个人脑补，但真实的影射了社会上所有的“主角”们，最后结局如何，只“主角”如何选择。

## 起因

不管生活如何，人会慢慢长大和老去，我总想留下些什么，某一天突然想到一个念头：如果未来的我可以知道现在的我都在做些什么，这种感觉肯定特别好吧，怀念。

于是想通过写月总结的形式，将当下的经历送给未来的自己，同时在每月总结中制定下个月的目标。

该 repo 目前存放着从我大三到现在的所有月总结。
